# Disaster Alleviation Foundation

Overview
The Disaster Alleviation Foundation (DAF) is a web-based application designed to streamline disaster management, volunteer coordination, and goods donations. This application allows users to:  

- View ongoing disasters and updates.  
- Register and manage volunteers.  
- Submit and track goods donations.  
- Monitor system performance under different loads.  

This project was developed as part of the Final Portfolio of Evidence (POE)for academic purposes.

 Features
- Disaster Management:Track disasters with relevant information and updates.  
- Volunteer Management: Add, update, and manage volunteer information.  
- Goods Donations:Submit and track donations of goods to support disaster relief.  
- Testing: Includes unit, integration, load, and stress tests to ensure system reliability.  
- Deployment: Application is deployed to Microsoft Azure using App Service for web hosting.  

Technologies Used
- **Frontend & Backend: ASP.NET Core MVC (.NET 8.0)  
- Database:SQL Server  
- Testing Frameworks: MSTest, JMeter  
- Deployment: Microsoft Azure App Service
